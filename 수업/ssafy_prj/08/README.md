  2. 성능차이가 나는 이유

  코드가 대체적으로 비슷한 편인데도 총접속자와 동시 접속자가 동일한 경우에도 평균 RPS와 응답시간의 편차가 컸다.
  상대방 컴퓨터와의 성능 차이도 있었겠지만 살펴보니 내가 bubble sort를 하는 코드를 추가하고 평균을 구하는 코드 대신 해당 코드를 실행시켜서 일어난 차이였다. bubble sort를 실행하다 보니 총 접속자와 동시접속자가 조금이라도 많아지면 fail 확률이 급격하게 늘어나는것을 볼 수 있었다. 실패율 또한 avg를 구하는 코드를 실행하도록 url을 바꾸고 나니 바로 수정이 되었다.
