## 01. 사용시 주의 사항

> git pull 전용입니다.

1. 해당 폴더 내에서는 어떠한 작업도 진행하지 않습니다.
2. clone 혹은 pull 받은 파일들은 개인 작업 공간으로 이동 후 사용합니다.

## 02. 사용 방법

- git clone 

```bash
# ~/Desktop
$ git clone https://lab.ssafy.com/s10/d01/ssafy-10th-gm1.git
```

- git pull

```bash
# ~/Desktop/ssafy-10th-gm1 (master)
$ git pull origin master
```
